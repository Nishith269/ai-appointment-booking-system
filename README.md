# AI Appointment Booking System

## 🚀 Overview
A full-stack AI-powered appointment booking system using LangGraph, FastAPI, and Streamlit.  
Supports booking, cancellation, listing, and querying appointments via natural language or structured UI.

---

## 🧠 Features
- Multi-agent workflow using LangGraph
- Tool-based execution (book, cancel, list, next slot)
- Streamlit UI with interactive controls
- FastAPI backend for scalable architecture
- Stateful conversations with memory

---

## 🏗 Architecture
Frontend (Streamlit) → FastAPI Backend → LangGraph Agent → Tools

---

## ⚙️ Tech Stack
- Python
- LangChain
- LangGraph
- FastAPI
- Streamlit
- Google Gemini API
